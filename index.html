📱 换设备教程
💡 数据保存在本设备本账号，换设备只需导出→导入，5步搞定
1
在当前设备：点右上角头像 → 「📦 导出我的数据」
会下载一个 JSON 备份文件，里面包含你全部的历史图纸和设置
2
把 JSON 文件传到新设备
U盘、微信文件传输助手、AirDrop、云盘都可以，随你喜欢
3
在新设备打开 BeadStudio，创建账号
建议用同一个昵称，方便记忆；PIN码可以和原来一样也可以重新设
4
登录后点右上角头像 → 「📥 导入数据」，选择 JSON 文件
稍等片刻，全部数据就会恢复
5
历史记录、图片库、标签、留言全部恢复 ✓
旧设备的数据还在，两边都可以继续用
